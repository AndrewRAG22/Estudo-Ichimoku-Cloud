📘 Miniguia de Estudo

📌 1. Resumo Estruturado

O Ichimoku Cloud é um indicador técnico composto por cinco elementos principais:
Tenkan-sen: média de curto prazo (9 períodos)
Kijun-sen: média de médio prazo (26 períodos)
Senkou Span A: média entre Tenkan e Kijun projetada no futuro
Senkou Span B: média de longo prazo (52 períodos) projetada no futuro
Chikou Span: preço atual deslocado para o passado

A área entre Senkou Span A e B forma a chamada nuvem (Kumo).

📈 Interpretação do Indicador
Preço acima da nuvem → tendência de alta
Preço abaixo da nuvem → tendência de baixa
Preço dentro da nuvem → mercado lateralizado

📌 2. Componentes e Funções
Tenkan-sen: indica mudanças rápidas de tendência
Kijun-sen: funciona como suporte/resistência
Kumo (Nuvem): identifica tendência e força do mercado
Chikou Span: confirma movimentos

📌 3. Estratégias e Sinais
Cruzamento Tenkan/Kijun (TK Cross):
Para cima → compra
Para baixo → venda
Rompimento da Nuvem:
Indica possível início de tendência
Sanyaku Kouten (Confirmação Tripla):
Preço acima da nuvem
Cruzamento de alta
Chikou confirmando

📌 4. Limitações
Muitos sinais falsos em mercados laterais
Pode ser complexo para iniciantes
Necessita prática para interpretação correta

📌 5. Uso Complementar
O Ichimoku pode ser combinado com:
RSI (Índice de Força Relativa)
Volume
Price Action

📖 Glossário

Ichimoku Kinko Hyo: gráfico de equilíbrio em um relance

Kumo: nuvem que indica tendência

Tenkan-sen: linha de conversão (curto prazo)

Kijun-sen: linha base (equilíbrio)

Chikou Span: linha atrasada

TK Cross: cruzamento entre Tenkan e Kijun

Kumo Twist: mudança de cor da nuvem indicando possível reversão
